2017-rapdev-backend README txt file

Requirements:
 - Python 2.7
 - click==6.7
 - Flask==0.12
 - Flask-SQLAlchemy==2.1
 - itsdangerous==0.24
 - Jinja2==2.9.5
 - MarkupSafe==0.23
 - PyJWT==1.4.2
 - SQLAlchemy==1.1.5
 - Werkzeug==0.11.15

Most requirements are installed with "pip install flask"

Start up Instructions:

1. run "python setup.py". This will set up a db ready for testing.
2. run "python main.py"
3. go to the printed out port in the terminal
4. $Profit$
